== Frequently Asked Questions ==

= How to Customize Theme =

Just click on Appearance -> customize
Then you will find section to edit and update it

= How to Customize Header Phone And Email =

Just click on Appearance -> Customize -> CBusiness Consult Header Phone and Email.

= How to Customize Header Social Link =

Just click on Appearance -> Customize -> CBusiness Consult Social Link.

= How to Customize Header Home banner =

Just click on Appearance -> Customize -> Header Image

= How to Customize Header Home banner heading and subheading =

Just click on Appearance -> Customize -> CBusiness Consult Lite Home banner Text

= How to Customize footer Copyright =

Just click on Appearance -> Customize -> CBusiness Consult Lite Footer

= How to Customize footer Widget =

Just click on Appearance -> Customize -> Widget
